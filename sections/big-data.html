- Paradigm shifts often ushered in by new instruments - Old: "vertical" research, many small-isolated teams, own data - New: "horizontal" integrate open data collected using many techniques - SDSS, New Mexico (2000-pres) - Big data astronomy before anyone was using the term big data - LHC Computing Grid (fast)

The era of "Brain Observatories"

  • Human Connectome Project (HCP),
    (2012-2015), N = ~1,200
  • Healthy Brain Network (HBN),
    (2016-pres),   N = ~10,000
  • Adolescent Brain Cognitive Development,
    (2016-pres),   N = ~10,000
  • UK Biobank,
    (2018-pres),   N = ~100,000
- HCP: paradigmatic dataset for shift to big data. (like Sloan) - HBN: recruiting 10,000 brains, pediatric mental health and learning disorders - ABCD: recruiting 10,000 brains, adolescent development - Biobank: recruiting 100,000, Alzheimers and others
